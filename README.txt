Flirtify — Quick deploy guide
-----------------------------

Files to upload:
- index.html
- privacy.html
- terms.html
- marketing_posts.txt

Quick publish options:

1) GitHub Pages (recommended)
   - Create a new public repo named "flirtify".
   - Upload the files to the repo root.
   - Settings → Pages → Source: Branch "main" (root) → Save.
   - Wait 1–5 minutes, your site will be available at: https://YOUR_GITHUB_USERNAME.github.io/flirtify

2) Netlify (very easy)
   - Go to https://app.netlify.com/drop
   - Drag & drop the files (or zip) to deploy instantly with HTTPS.

3) Vercel
   - Link your GitHub account, import the repository and deploy.

Indexing on Google:
- After site is live, open Google Search Console → Add property → URL inspection → Request indexing.

Tips:
- Use the marketing posts in 'marketing_posts.txt' to create TikToks and Instagram posts.
- Monitor clicks and conversions (you can add Google Analytics later).
